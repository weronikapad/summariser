# Summariser

Summarises websites and pdf content

## Key features
- fetches text from pdfs and websites
- cuts the text to 10000 chars nad feeds it to groq to summarise
- streamlit web UI

# Tech Stack
Python 3, Streamlit, Groq API, BeautifulSoup4, PyPDF, Requests.

# How to use
1. Go to this website https://summariser-4cdu.onrender.com/

2. Paste a URL (make sure it html) or upload a pdf file that you want to summarise

3. Click the summarise button

4. If you want to run it again delete the url/pdf that you used previously upload the new url/pdf and click summarise

disclaimer: It might take a few seconds for the program to print your summarisation
